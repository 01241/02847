# 0-1-Box

Sounds simple: Board with two Motion Sensors.
"Use case": Mobile pollbox YES|NO

## Objective:
* Send the impuls from the activated sensor to a website via LoraWAN
* Save the data in GoogleSpreadsheet
* Visualization
